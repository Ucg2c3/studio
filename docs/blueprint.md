# **App Name**: CloudGenius

## Core Features:

- Prototype Generation: Use Claude AI to generate a prototype app based on user input. The AI will act as a tool to decide on the optimal layout, components and features based on the user's specifications.
- Prototype Display: Display the generated prototype in a structured, user-friendly interface, showcasing the app's layout and components.
- Export to Next.js: Allow users to export the prototype as a Next.js project, including all necessary files and configurations.

## Style Guidelines:

- Primary color: Deep sky blue (#007BFF) for a modern and clean feel.
- Secondary color: Light gray (#F8F9FA) for backgrounds and subtle contrasts.
- Accent: Emerald green (#28A745) for success states and primary actions.
- Clean and modern typography for readability and a professional look.
- Use a grid-based layout for a structured and organized presentation of the prototype.
- Simple and intuitive icons to represent different components and actions.
